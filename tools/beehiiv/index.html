Yes, Beehiiv has a free plan supporting up to 2500 subscribers. This makes it accessible for new senders testing the platform.

Beehiiv is best for newsletter publishers and media operators. Its standout features include newsletter monetisation, ad network, referral program, which suit this use case well.

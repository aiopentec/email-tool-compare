Frequently asked questions

Is Buttondown free?

Yes, Buttondown has a free plan supporting up to 100 subscribers. This makes it accessible for new senders testing the platform.

What is Buttondown best used for?

Buttondown is best for developers and technical writers who prefer markdown. Its standout features include markdown writing, API access, paid subscriptions, which suit this use case well.

Is Buttondown good for beginners?

Buttondown is generally beginner-friendly. The interface is straightforward for basic campaigns, though advanced automation features have a learning curve.

Does Buttondown have an affiliate program?

Buttondown does not currently offer an affiliate program. This is worth considering if you plan to recommend the platform to your audience.
